How to install?
===============

```
$ git clone --recursive git@github.com:mahendrakalkura/pinbet88.com.git
$ cd pinbet88.com
$ mkvirtualenv pinbet88.com
$ pip install --requirement requirements.txt
```

How to run?
===========

```
$ cd pinbet88.com
$ workon pinbet88.com
$ USERNAME=... PASSWORD=... python manage.py --matches
```
